# Transferuri izocrone - USB

- `isos` - egal, uniform; `chronos` - timp
- `Izocron` = cu durata egala; care apare la intervale egale
- se utilizeaza pentru datele generate in timp real, care trebuie furnizate cu rata cu care sunt receptionate
- trebuie respectata si intarzierea maxima cu care sunt furnizate datele

Furnizarea la timp a datelor este asigurata cu pretul unor pierderi potentiale in sirul de date.

Portiune dedicata a latimii de banda

Transferuri `izocrone`:
    - furnizarea la timp a datelor
    - lipsa retransmiterii datelor eronate

Transferuri `asincrone`:
    - furnzarea corecta a datelor
    - retransmiterea datelor eronate
