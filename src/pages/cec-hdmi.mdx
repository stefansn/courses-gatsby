# CEC - interfata HDMI

Este o magistrala seriala bidirectionala, cu un fir pentru transferul codurilor telecomenzilor.
- One touch play, system standby, tuner control

Utilizatorul poate controla mai multe aparate conectate prin HDMI cu o telecomanda.

Aparatele se pot controla unele pe altele, fara interventia utilizatorului.
