# Comparatie placi video integrate si dedicate

**GPU dedicate**

- utilizate in placile grafice
- contin o memorie grafica dedicata pentru GPU
- comunica cu UCP prin magistrala sistem, de obicei, o magistrala PCI Express.

**GPU integrate**

- integrate intr-un set de circuite sau circuit SoC
- utilizeaza o parte a memoriei sistem
- performante mai reduse fata de GPU dedicate
