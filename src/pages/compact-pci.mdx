# Compact PCI

**Scop**: inlocuire magistralei `VME` cu `PCI`

Module `CompactPCI` originale:
- magistrala PCI paralele de 32 biti sau 64 biti
- placi de baza

**Prezentare generala**

`Placi de baza pasive` -> intretinere simpla.

Numar mai mare de socluri decat in sistemele PCI conventionale
- `Conectori cu pini si socluri` de calitate ridicata
- Numar mare de `pini de masa`

Conectori:
-`HM` (Hard Metric); pas de 2mm

**Extensii CompactPCI**

Permit utilizarea sistemelor `CompactPCI` in aplicatii variate.

**Hot Swap**

Defineste cerinte suplimentare pentru adaugare si eliminarea modulelor in timpul functionarii.

Sunt prevazuti cu pini cu lungime multiple.

**Conectivitate Ethernet**

Permite crearea unei retele locale intre module de extensie multiple.

Max. 2 Gbiti/s pentru fiecare soclu CompactPCI

<h3>Avantaje CompactPCI</h3>

- modularitate, robustete
- scalabilitate
- independenta fata de procesor
- suport extins software si SO, compatibilitate cu drivere si aplicatii existente
- cost redus al circuitelor PCI si PCIe
- extensiile specificatiei de baza permit tranzitia la interconexiuni seriale
